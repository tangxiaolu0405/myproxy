# myproxy - 轻量级代理客户端

基于 Fyne 的跨平台桌面应用，集成 xray-core 作为代理引擎，提供直观的 GUI 界面管理代理服务。

## 主要功能

- **图形界面**：订阅管理、服务器列表、延迟测试、代理控制、实时日志
- **代理引擎**：内置 xray-core，支持 SOCKS5/VMess 协议，默认开启本地 10808 端口
- **订阅支持**：兼容 VMess、SOCKS5、JSON/Base64 格式订阅
- **数据持久化**：服务器和配置存储在 SQLite 数据库
- **主题与布局**：支持浅色/深色主题，窗口布局自动保存

## 快速开始

### 环境要求
- Go 1.25.4+
- macOS / Windows / Linux（需图形环境）

### 安装与运行

```bash
go mod download
go run ./cmd/gui/main.go

# 或编译后运行
go build -o gui ./cmd/gui
./gui
```

首次启动会自动创建数据库并初始化配置。

## 基本使用

1. **添加订阅**：在订阅面板输入订阅 URL，可添加标签
2. **管理服务器**：等待订阅解析完成后，在服务器列表中查看和管理
3. **测试延迟**：点击服务器列表中的测试按钮
4. **启动代理**：选择服务器后点击启动按钮，默认使用 10808 端口
5. **查看日志**：实时查看应用和代理日志

## 配置说明

- **默认端口**：10808（SOCKS5）
- **数据库**：`./data/myproxy.db`
- **日志文件**：`myproxy.log`

## 技术架构

- **GUI**：Fyne 框架
- **代理**：xray-core（库方式集成）
- **存储**：SQLite 数据库
- **订阅**：支持多种格式解析
